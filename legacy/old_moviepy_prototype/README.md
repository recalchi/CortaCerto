# Protótipo legado MoviePy

Esta pasta preserva os scripts e anotações da fase inicial do CortaCerto baseada em MoviePy/pydub.

O app atual não usa estes arquivos como entrada principal. Use:

```bat
python main.py
```

ou:

```bat
run.bat
```

A suíte oficial da sprint fica em `tests/` e deve ser rodada com:

```bat
python scripts\run_sprint_checks.py
```
